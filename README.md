## Features

- **Dynamic Interface** — A highly interactive, fullpage scrolling experience with smooth Framer Motion page transitions and staggered entrance animations.
- **Project Showcase** — A detailed and filterable grid of my Web Development, AI/ML, and freelance projects.
- **Generative Engine Optimization (GEO)** — Built with modern SEO and GEO principles, utilizing Next.js App Router for optimal indexability and performance.
- **Interactive Widgets** — Includes real-time Spotify "now playing" integrations and deferred chat widget loading.
- **Fully Responsive** — Beautifully scales across mobile, tablet, and desktop viewports.

## Tech Stack

This portfolio was built using modern web technologies:

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion & GSAP
- **Data Management:** JSON-driven architecture
- **Deployment:** GitHub Pages (Static Export)
- **Analytics:** Vercel Analytics
